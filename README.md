# Adam Vue Portfolio

This is my personal portfolio website showcasing my experience, projects, and coursework. It was built as a static HTML/CSS site with a black-and-sage aesthetic.

## Features

- **Hero Section:** Introduction with a short bio and image.
- **Experience Section:** Highlights internships and relevant work experience.
- **Projects Section:** Displays notable software projects.
- **Courses Section:** Lists coursework with brief descriptions.
- **Responsive Navigation:** Navbar links to Home and Courses pages.
- **Favicon:** Custom SVG icon for browser tabs.

## Technologies Used

- HTML5
- CSS3
- Google Fonts: [Cormorant](https://fonts.google.com/specimen/Cormorant), [Libre Caslon](https://fonts.google.com/specimen/Libre+Caslon+Text)
